# Lista de Tarefas

Aplicação simples de lista de tarefas desenvolvida com HTML, CSS e JavaScript puro.

O projeto permite adicionar, editar, remover e limpar tarefas dinamicamente utilizando manipulação do DOM e arrays em JavaScript.

## Funcionalidades

- Adicionar tarefas
- Editar tarefas
- Remover tarefas
- Limpar toda a lista
- Layout responsivo
- Interface estilizada com CSS
- Atualização dinâmica da lista sem recarregar a página

## Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript (Vanilla JS)

## Conceitos Praticados

Durante o desenvolvimento deste projeto foram utilizados conceitos importantes de front-end, como:

- Manipulação do DOM
- Arrays
- Funções
- Eventos (`onclick`)
- Estruturas de repetição (`for`)
- Condicionais (`if/else`)
- Métodos de array (`push`, `splice`)
- Criação dinâmica de elementos (`createElement`)
- Atualização de conteúdo com `textContent`
- Estilização responsiva com CSS
